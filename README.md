# AI Interview Assistant

AI 기반 실시간 면접 연습 서비스입니다. 이력서/자기소개서를 업로드하면 AI가 맞춤형 면접 질문을 생성하고, 음성으로 실시간 면접을 진행하며, 결과 리포트까지 제공합니다.

---

## 주요 기능

### 사용자 기능

#### A. 면접 준비
- 학과 선택
- 면접 유형 선택 (인성 / 전공 / 자기소개서 기반 / 혼합)
- 면접관 스타일 선택 또는 랜덤 (부드러운 / 압박 / 꼼꼼한 교수형 / 실무형)

#### B. 자료 업로드
- 이력서 / 자기소개서 업로드
- 업로드 시 개인정보 자동 검사 (주민번호, 상세주소, 연락처, 이메일, 생년월일 등)
- 민감정보 발견 시 업로드 제한 및 재업로드 안내

#### C. AI 분석
- 이력서/자소서 핵심 키워드 추출
- 프로젝트 경험 및 기술 스택 추출
- 전공지식 연계 예상 질문 생성
- 약점 질문 후보 생성

#### D. 실시간 면접
- AI 질문 TTS 음성 출력
- 사용자 답변 음성 입력 → STT 변환
- 답변 기반 꼬리질문 자동 생성
- 면접관 캐릭터 말투 반영

#### E. 면접 결과 리포트
- 답변 내용 / 논리성 / 전공지식 점수
- 말하기 속도, 음성 떨림, 무음 구간 분석
- 표정 / 시선 / 긴장도 관찰 피드백
- 부족한 CS/전공 개념 추천
- 반복 연습 모드 (오답 기반 약점 학습)

---

### 관리자 기능

#### A. 학과 관리
- 학과 등록 / 수정 / 삭제
- 학과별 전공지식 문서 등록 및 과목별 분류 (자료구조, DB, 운영체제, 네트워크, 웹프로그래밍, Java/C 등)

#### B. 질문 데이터 관리
- 전공지식 기반 예상 질문 자동 생성
- 관리자 검수 후 공개
- 난이도 설정 (기초 / 중급 / 심화)

#### C. 피드백 데이터 관리
- 특정 질문에 대한 모범 답안 등록
- 자주 부족한 개념 연결
- 추천 학습 자료 연결

## 기술 스택

### 프론트엔드
- **언어**: TypeScript
- **프레임워크**: Next.js 16.2.1 (React 19 기반)
- **스타일링**: Tailwind CSS v4
- 웹캠 / 마이크 접근 (Web Audio API, MediaPipe)
- 실시간 면접 화면 UI
- 면접 결과 대시보드

### 백엔드
- Node.js + Express / NestJS
- 사용자 인증
- 파일 업로드 처리
- AI API 호출
- 면접 세션 관리 및 결과 저장

### 데이터베이스
- PostgreSQL 또는 MySQL
- 사용자 정보, 업로드 파일 메타데이터
- 학과 / 전공지식 / 질문 / 답변 기록
- 면접 결과 리포트 저장

### AI / 미디어
| 기능 | 기술 |
|------|------|
| 질문 생성 / 답변 분석 / 피드백 생성 | GPT (OpenAI) |
| 질문 음성 출력 | TTS (OpenAI) |
| 사용자 답변 텍스트 변환 | STT (AWS Transcribe) |
| 실시간 양방향 음성 | OpenAI Realtime API |
| 표정 / 시선 분석 | MediaPipe Face Landmarker |
| 음성 지표 분석 | Web Audio API |

## AI 질문 생성 로직

단순히 GPT에 질문 생성을 맡기는 것이 아니라, 아래 입력 데이터를 조합하여 품질 높은 질문을 생성합니다.

**입력 데이터**
- 선택 학과 / 관리자 등록 전공지식
- 이력서 / 자소서 내용 / 사용 기술 스택
- 면접 유형 / 난이도 / 이전 답변 내용

**질문 4가지 유형**
1. **자소서 기반** - 프로젝트 경험, 기술 선택 이유, 어려웠던 점
2. **전공지식** - CS 개념, 자료구조, DB, 네트워크 등
3. **꼬리질문** - 직전 답변 기반 심화 질문
4. **압박/예외 질문** - 실패 경험, 갈등 처리, 기여도 검증

## 피드백 원칙

### 표정/시선 분석
- "자신감이 없어 보입니다" 같은 단정적 표현 지양
- 관찰 기반 피드백으로 제공
  - "면접 중 표정 변화가 적게 관찰되었습니다"
  - "정면 응시 시간이 짧았습니다"

### 음성 분석 지표
- 평균 말하기 속도 (WPM)
- 무음 구간 수 / 답변 시작 전 지연 시간
- 필러어 사용 빈도 ("음…", "어…", "그…")
- 음량 변화 폭

## 안전장치

- **개인정보 필터링**: 업로드 파일에서 전화번호, 이메일, 주민번호, 상세주소 자동 감지 후 마스킹/반려 처리
- **분석 결과 표현 주의**: 표정/음성 피드백은 의학적 진단이 아닌 면접 전달력 관점으로 제한
- **관리자 검수**: 전공지식 기반 질문은 자동 생성 후 관리자 검수 과정을 거쳐 반영
