## FE-001 메인 랜딩 페이지 (`/`)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [Navbar] 로고 · 로그인 버튼             │
├─────────────────────────────────────────┤
│  [Hero Section]                          │
│   타이핑 애니메이션 타이틀               │
│   서비스 소개 문구                       │
│   [면접 시작하기] CTA 버튼              │
├─────────────────────────────────────────┤
│  [기능 소개 카드]                        │
│   AI 질문 생성 · 실시간 면접 · 결과 리포트│
│   (스크롤 reveal 애니메이션)            │
└─────────────────────────────────────────┘
```

### 주요 컴포넌트

| 컴포넌트 | 기능 |
|----------|------|
| Navbar | 상단 네비게이션 바 (로고, 로그인 버튼) |
| TypewriterText | 타이핑 효과 타이틀 |
| StartButton | CTA 버튼 → `/login` 이동 |
| ScrollReveal | 스크롤 시 카드 fade-in 애니메이션 |

### 화면 흐름

- [면접 시작하기] → `/login`

## FE-002 로그인 페이지 (`/login`)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [로그인 탭] 일반 로그인 | 관리자 로그인  │
├─────────────────────────────────────────┤
│  이메일 입력 필드                        │
│  비밀번호 입력 필드                      │
│  [로그인] 버튼                          │
│  회원가입 링크                          │
└─────────────────────────────────────────┘
```

### 입력 항목

| 필드 | 타입 | 필수 | 유효성 검사 |
|------|------|------|------------|
| 이메일 | text | ✓ | 이메일 형식 |
| 비밀번호 | password | ✓ | 8자 이상 |

### 화면 흐름

- 일반 로그인 성공 → 세션(localStorage) 저장 → `/setup`
- 관리자 로그인 성공 → 세션(localStorage) 저장 → `/admin`
- 로그인 실패 → 에러 메시지 표시

## FE-003 회원가입 페이지 (`/signup`)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  이름 입력 필드                          │
│  이메일 입력 필드                        │
│  비밀번호 입력 필드                      │
│  비밀번호 확인 입력 필드                 │
│  [회원가입] 버튼                        │
│  로그인 링크                            │
└─────────────────────────────────────────┘
```

### 입력 항목

| 필드 | 타입 | 필수 | 유효성 검사 |
|------|------|------|------------|
| 이름 | text | ✓ | 공백 불가 |
| 이메일 | text | ✓ | 이메일 형식, 중복 불가 |
| 비밀번호 | password | ✓ | 8자 이상 |
| 비밀번호 확인 | password | ✓ | 비밀번호 일치 |

### 화면 흐름

- 회원가입 성공 → `/login`

## FE-004 면접 설정 페이지 (`/setup`)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [StepNavbar] 1.설정 > 2.업로드 > 3.면접│
├─────────────────────────────────────────┤
│  [Step 1] 학과 선택 (그리드)             │
│    활성: 컴퓨터소프트웨어과              │
│    비활성: 나머지 학과 (coming soon)     │
├─────────────────────────────────────────┤
│  [Step 2] 직무 선택 (드롭다운 / 그리드)  │
│    학과별 직무 목록 표시                 │
├─────────────────────────────────────────┤
│  [Step 3] 면접 유형 선택                │
│    인성 · 직무 · 전공 · 이력서 · 혼합   │
├─────────────────────────────────────────┤
│  [Step 4] 기업 유형 선택                │
│    스타트업 · 중소기업 · 중견기업 · 대기업│
│    공기업 · 외국계                      │
├─────────────────────────────────────────┤
│  [Step 5] 경력 수준 선택                │
│    신입 · 주니어 · 미드레벨 · 시니어    │
├─────────────────────────────────────────┤
│  [Step 6] 면접 스타일 선택              │
│    친절 · 압박 · 교수형 · 실무형 · 랜덤 │
├─────────────────────────────────────────┤
│  [다음] 버튼 → /upload                  │
└─────────────────────────────────────────┘
```

### 선택 항목

| 항목 | 선택지 | 필수 |
|------|--------|------|
| 학과 | 컴퓨터소프트웨어과 (외 비활성) | ✓ |
| 직무 | 학과별 직무 목록 | ✓ |
| 면접 유형 | 인성/직무/전공/이력서/혼합 | ✓ |
| 기업 유형 | startup/smb/midsize/large/public/foreign | ✓ |
| 경력 수준 | newcomer/junior/mid/senior | ✓ |
| 면접 스타일 | friendly/pressure/professor/practical/random | ✓ |

## FE-006 면접 진행 페이지 (`/interview`)

### 화면 구성 — 준비 화면

```
┌─────────────────────────────────────────┐
│  카메라·마이크 장치 선택 드롭다운        │
│  화면 레이아웃 선택                      │
│    분할화면 / PiP / 면접관 전체화면      │
│  [면접 시작] 버튼                       │
│  (질문 생성 로딩 프로그레스 바)         │
└─────────────────────────────────────────┘
```

### 화면 구성 — 면접 진행 화면

```
┌───────────────────────┬─────────────────┐
│   AI 면접관 아바타     │   응시자 카메라  │
│   (스타일별 이미지)    │   (실시간 영상)  │
│   [Wav2Lip 립싱크 영상]│                 │
├───────────────────────┴─────────────────┤
│  [질문 텍스트 + 카테고리 배지]           │
│  질문 번호 · 진행 타이머                 │
├─────────────────────────────────────────┤
│  [녹음 컨트롤]                          │
│    [● 녹음 시작] / [■ 녹음 종료]        │
│    마이크 레벨 시각화 바                 │
│    남은 시간 표시 (최대 90초)           │
└─────────────────────────────────────────┘
```

### 화면 구성 — 답변 검토 화면

```
┌─────────────────────────────────────────┐
│  [STT 변환 결과 텍스트 편집 영역]        │
├─────────────────────────────────────────┤
│  [재답변] [수정 완료] [다음 질문 →]     │
└─────────────────────────────────────────┘
```

### 얼굴 분석 정보 (오버레이)

| 항목 | 표시 방식 |
|------|----------|
| 시선 방향 | 정면/좌/우/위/아래 텍스트 |
| 눈 깜빡임 횟수 | 숫자 카운터 |
| 분석 주기 | 200ms |

## FE-009 관리자 페이지 (`/admin`)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  학과 선택 드롭다운                      │
├─────────────────────────────────────────┤
│  [전공지식 목록 테이블]                  │
│    과목 · 내용 · 등록자 · 등록일 · 삭제 │
├─────────────────────────────────────────┤
│  [항목 추가 폼]                         │
│    학과 · 과목 · 내용 · 등록자          │
│    [등록] 버튼                          │
└─────────────────────────────────────────┘
```

### 접근 제한

- 관리자(`role=admin`) 계정으로 로그인한 경우에만 접근 가능
- 일반 사용자 접근 시 `/login` 리다이렉트
