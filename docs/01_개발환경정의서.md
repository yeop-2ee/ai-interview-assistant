## 5. 환경변수 목록

### Frontend (`frontend/.env.local`)

| 변수명 | 기본값 | 설명 |
|--------|--------|------|
| NEXT_PUBLIC_BACKEND_URL | http://localhost:3001 | 백엔드 API URL |
| NEXT_PUBLIC_MEDIA_API | http://localhost:4000 | 미디어 서비스 URL |

### Backend (`backend/.env`)

| 변수명 | 예시 | 설명 |
|--------|------|------|
| DATABASE_URL | postgresql://user:pw@localhost:5432/ai_interview | PostgreSQL 연결 URL |
| JWT_SECRET | (임의 문자열) | JWT 서명 키 |
| AI_SERVER_URL | http://localhost:5050 | AI Pipeline URL |
| MEDIA_SERVER_URL | http://localhost:4000 | 미디어 서비스 URL |
| PORT | 3001 | 서버 포트 |

### AI Pipeline (`ai-pipeline/.env`)

| 변수명 | macOS 기본값 | Windows 기본값 | 설명 |
|--------|-------------|----------------|------|
| PORT | 5050 | 5050 | 서버 포트 |
| MLX_SERVER_URL | http://localhost:8080 | - | MLX LM 서버 URL |
| MLX_MODEL | mlx-community/Meta-Llama-3.1-8B-Instruct-4bit | - | MLX 모델명 |
| LLM_SERVER_URL | - | http://localhost:11434 | Ollama URL |
| LLM_MODEL | - | llama3.1:8b | Ollama 모델명 |

### Media Service (`media-service/.env`)

| 변수명 | 기본값 | 설명 |
|--------|--------|------|
| PORT | 4000 | 서버 포트 |
| PYTHON_PATH | python3 (macOS) / python (Windows) | Python 실행 경로 |
| WHISPER_MODEL | mlx-community/whisper-large-v3-turbo (macOS) / large-v3-turbo (Windows) | Whisper 모델 |
| WAV2LIP_INFERENCE_PATH | (사용자 설정) | Wav2Lip inference.py 경로 |
| WAV2LIP_CHECKPOINT_PATH | (사용자 설정) | Wav2Lip 체크포인트 경로 |

## 6. 실행 순서

```
1. PostgreSQL 기동
2. LLM 서버 기동 (MLX / Ollama)
3. AI Pipeline 서버 기동 (node aiServer.js)
4. Media Service 서버 기동 (npm start)
5. Backend 서버 기동 (npx ts-node src/server.ts)
6. Frontend 개발 서버 기동 (npm run dev)
```
